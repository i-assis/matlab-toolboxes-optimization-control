O acesso se dá pelo comando:
\begin{verbatim}
	ssh labcamino@caminolab9.fem.unicamp.br
\end{verbatim}
onde ``labcamino'' é o usuário e o trecho após o ``\verb|@|'' é o endereço da servidora.

No primeiro \textit{login} é necessário inserir a senha.
Todos os usuários do laboratório sabem essa senha.

O comando ``cd'' entra nas pastas. Utilize-o para chegar no local onde o arquivo se encontra \footnote{No ``DOS'' do Windows pode-se copiar o endereço na janela e colar clicando com o botão direito. Não sei se isso funciona no ``Linux''}. Verifique o resultado na \autoref{fig_05}.

Para enviar um arquivo deve-se utilizar ``scp'':
\begin{verbatim}
scp arq.ext destino:. Enter
\end{verbatim}

Para enviar um arquivo nomeado ``exemplo.tex'' para a servidora:
\begin{verbatim}
scp exemplo.tex labcamino(at)caminolab9.fem.unicamp.br:.
\end{verbatim}

Repare que os dois pontos ``:`` e o ponto ``.'' devem ser adicionados\footnote{Não sei quais as combinação possíveis}.

\begin{verbatim}
scp -pr NOMEDAPASTA/ destino:. Enter
\end{verbatim}
